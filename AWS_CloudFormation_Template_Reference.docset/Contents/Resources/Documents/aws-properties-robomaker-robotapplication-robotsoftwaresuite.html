RoboMaker RobotApplication RobotSoftwareSuite

The RobotSoftwareSuite property type specifies the robotsoftware suite for an AWS RoboMaker robot application.

RobotSoftwareSuite is a property of the AWS::RoboMaker::RobotApplication resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Name" : String,
  "Version" : String
}

YAML

Name: String
Version: String

Properties

Name

The name of the robot software suite.

Required: Yes

Type: String

Update requires: No interruption

Version

The version of the robot software suite.

Required: Yes

Type: String

Update requires: No interruption

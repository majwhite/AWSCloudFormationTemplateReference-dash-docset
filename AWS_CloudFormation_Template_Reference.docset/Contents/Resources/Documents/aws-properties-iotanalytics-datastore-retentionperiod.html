AWS IoT Analytics Datastore RetentionPeriod

The RetentionPeriod property type specifies how log, in days, message data is kept for an AWS IoT Analytics datastore.

RetentionPeriod is a property of the AWS::IoTAnalytics::Datastore resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "NumberOfDays" : Integer,
  "Unlimited" : Boolean
}

YAML

NumberOfDays: Integer
Unlimited: Boolean

Properties

NumberOfDays

The number of days that message data is kept. The "unlimited" parameter must be false.

Required: No

Type: Integer

Update requires: No interruption

Unlimited

If true, message data is kept indefinitely.

Required: No

Type: Boolean

Update requires: No interruption

ConfigurationAssociation ConfigurationId

The ConfigurationId property type specifies a configuration Id and the revision of a configuration.

ConfigurationId is a property of the AWS::AmazonMQ::ConfigurationAssociation resource type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Id" : String,
  "Revision" : Integer
}

YAML

Id: String
Revision: Integer

Properties

Id

The Id of the configuration.

Required: Yes

Type: String

Update requires: No interruption

Revision

The revision number of the configuration.

Required: Yes

Type: Integer

Update requires: No interruption

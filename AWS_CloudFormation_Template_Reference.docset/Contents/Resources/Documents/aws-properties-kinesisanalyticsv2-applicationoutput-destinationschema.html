Amazon Kinesis Data Analytics ApplicationOutput DestinationSchema

The DestinationSchema property type specifies the data format when records are written to the destination in an SQL-based Amazon Kinesis Data Analytics application.

DestinationSchema is a property of the Output property.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "RecordFormatType" : String
}

YAML


RecordFormatType: String

Properties

RecordFormatType

Specifies the format of the records on the output stream.

Required: No

Type: String

Update requires: No interruption

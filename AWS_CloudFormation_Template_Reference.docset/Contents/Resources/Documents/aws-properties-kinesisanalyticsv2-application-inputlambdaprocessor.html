Amazon Kinesis Data Analytics Application InputLambdaProcessor

The InputLambdaProcessor property type specifies the Amazon Resource Name (ARN) of the AWS Lambda function that is used to preprocess records in the stream in an SQL-based Amazon Kinesis Data Analytics application.

InputLambdaProcessor is a property of the InputProcessingConfiguration property.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "ResourceARN" : String
}

YAML


ResourceARN: String

Properties

ResourceARN

The ARN of the AWS Lambda function that operates on records in the stream.

Required: Yes

Type: String

Update requires: No interruption

AWS::Cognito::UserPool Policies

Policies is a property of the AWS::Cognito::UserPool resource that defines the password policies of an Amazon Cognito User Pool.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "PasswordPolicy" : PasswordPolicy
}

YAML

  PasswordPolicy: 
    PasswordPolicy

Properties

PasswordPolicy

The password policy.

Required: No

Type: PasswordPolicy

Update requires: No interruption

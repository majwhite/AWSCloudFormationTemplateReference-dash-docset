AWS::Cognito::IdentityPool PushSync

PushSync is a property of the AWS::Cognito::IdentityPool resource that defines the configuration options to be applied to an Amazon Cognito identity pool.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "ApplicationArns" : [ String, ... ],
  "RoleArn" : String
}

YAML

  ApplicationArns: 
    - String
  RoleArn: String

Properties

ApplicationArns

List of Amazon SNS platform application ARNs that could be used by clients.

Required: No

Type: List of String

Update requires: No interruption

RoleArn

An IAM role configured to allow Amazon Cognito to call SNS on behalf of the developer.

Required: No

Type: String

Update requires: No interruption

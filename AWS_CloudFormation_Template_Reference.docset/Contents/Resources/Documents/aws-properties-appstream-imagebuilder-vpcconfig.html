Amazon AppStream 2.0 ImageBuilder VpcConfig

The VpcConfig property type specifies the VPC configuration for an Amazon AppStream 2.0 image builder.

VpcConfig is a property of the AWS::AppStream::ImageBuilder resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "SecurityGroupIds" : [ String, ... ],
  "SubnetIds" : [ String, ... ]
}

YAML


SecurityGroupIds: 
  - String
SubnetIds: 
  - String

Properties

SecurityGroupIds

The identifiers of the security groups for the image builder.

Required: No

Type: List of String values

Update requires: No interruption

SubnetIds

The identifier of the subnet to which a network interface is attached from the image builder instance.

Required: No

Type: List of String values

Update requires: No interruption

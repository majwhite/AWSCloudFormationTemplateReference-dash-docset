Amazon Kinesis Data Analytics Application FlinkApplicationConfiguration

The FlinkApplicationConfiguration property type describes configuration parameters for a Java-based Kinesis Data Analytics application.

FlinkApplicationConfiguration is a property of the ApplicationConfiguration property.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "CheckpointConfiguration" : CheckpointConfiguration,
  "ParallelismConfiguration" : ParallelismConfiguration,
  "MonitoringConfiguration" : MonitoringConfiguration
}

YAML


CheckpointConfiguration: CheckpointConfiguration
ParallelismConfiguration: ParallelismConfiguration
MonitoringConfiguration: MonitoringConfiguration

Properties

CheckpointConfiguration

Describes an application's checkpointing configuration. Checkpointing is the process of persisting application state for fault tolerance.

Required: No

Type: CheckpointConfiguration

Update requires: No interruption

ParallelismConfiguration

Describes parameters for how an application executes multiple tasks simultaneously.

Required: No

Type: ParallelismConfiguration

Update requires: No interruption

MonitoringConfiguration

Describes configuration parameters for Amazon CloudWatch logging for an application.

Required: No

Type: MonitoringConfiguration

Update requires: No interruption

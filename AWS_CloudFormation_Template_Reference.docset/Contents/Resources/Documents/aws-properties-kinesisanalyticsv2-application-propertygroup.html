Amazon Kinesis Data Analytics Application PropertyGroup

The PropertyGroup property type specifies property key-value pairs passed into a Java-based Kinesis Data Analytics application.

PropertyGroup is a property of the EnvironmentProperties property.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "PropertyMap" : Json,
  "PropertyGroupId" : String
}

YAML


PropertyMap: Json
PropertyGroupId: String

Properties

PropertyMap

Describes the value of an application execution property key-value pair.

Required: No

Type: Json

Update requires: No interruption

PropertyGroupId

Describes the key of an application execution property key-value pair.

Required: No

Type: String

Update requires: No interruption

AWS::Cognito::IdentityPoolRoleAttachment RulesConfigurationType

RulesConfigurationType is a subproperty of the RoleMapping property that defines the rules to be used for mapping users to roles.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Rules" : [ MappingRule, ... ]
}

YAML

  Rules: 
    - MappingRule

Properties

Rules

A list of rules. You can specify up to 25 rules per identity provider.

Required: Yes

Type: List of MappingRule

Update requires: No interruption

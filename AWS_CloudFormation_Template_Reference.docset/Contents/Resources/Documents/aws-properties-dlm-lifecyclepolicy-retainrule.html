Amazon Data Lifecycle Manager LifecyclePolicy RetainRule

The RetainRule property type specifies a snapshot retention rule for an Amazon Data Lifecycle Manager lifecycle policy.

RetainRule is a property of the Schedule property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Count" : Integer
}

YAML

Count: Integer

Properties

Count

The number of snapshots to keep for each volume, up to a maximum of 1000.

Required: Yes

Type: Integer

Update requires: No interruption

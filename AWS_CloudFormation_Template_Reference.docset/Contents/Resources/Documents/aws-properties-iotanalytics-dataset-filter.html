AWS IoT Analytics Dataset Filter

The Filter property type specifies pre-filters which are applied to message data for an AWS IoT Analytics dataset.

Filter is a property of the QueryAction property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "DeltaTime" : DeltaTime
}

YAML

DeltaTime: 
  DeltaTime

Properties

DeltaTime

Used to limit data to that which has arrived since the last execution of the action.

Required: No

Type: DeltaTime

Update requires: No interruption

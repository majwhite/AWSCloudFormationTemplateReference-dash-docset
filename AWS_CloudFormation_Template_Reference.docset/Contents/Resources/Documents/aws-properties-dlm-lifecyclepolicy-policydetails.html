AWS::DLM::LifecyclePolicy PolicyDetails

Specifies the configuration of a lifecycle policy.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Parameters" : Parameters,
  "PolicyType" : String,
  "ResourceTypes" : [ String, ... ],
  "Schedules" : [ Schedule, ... ],
  "TargetTags" : [ Tag, ... ]
}

YAML

  Parameters: 
    Parameters
  PolicyType: String
  ResourceTypes: 
    - String
  Schedules: 
    - Schedule
  TargetTags: 
    - Tag

Properties

Parameters

A set of optional parameters that can be provided by the policy.

Required: No

Type: Parameters

Update requires: No interruption

PolicyType

This field determines the valid target resource types and actions a policy can manage. This field defaults to EBS_SNAPSHOT_MANAGEMENT if not present.

Required: No

Type: String

Allowed Values: EBS_SNAPSHOT_MANAGEMENT

Update requires: No interruption

ResourceTypes

The resource type.

Required: No

Type: List of String

Maximum: 1

Update requires: No interruption

Schedules

The schedule of policy-defined actions.

Required: No

Type: List of Schedule

Maximum: 1

Update requires: No interruption

TargetTags

The single tag that identifies targeted resources for this policy.

Required: No

Type: List of Tag

Maximum: 50

Update requires: No interruption

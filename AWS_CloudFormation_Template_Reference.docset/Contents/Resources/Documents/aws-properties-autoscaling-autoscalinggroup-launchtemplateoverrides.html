AWS::AutoScaling::AutoScalingGroup LaunchTemplateOverrides

LaunchTemplateOverrides is a subproperty of LaunchTemplate that describes an override for a launch template.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "InstanceType" : String
}

YAML

  InstanceType: String

Properties

InstanceType

The instance type. For more information, see Available Instance Types in the Amazon EC2 User Guide for Linux Instances.

Required: No

Type: String

Minimum: 1

Maximum: 255

Pattern: [\u0020-\uD7FF\uE000-\uFFFD\uD800\uDC00-\uDBFF\uDFFF\r\n\t]*

Update requires: No interruption

AWS::AppMesh::Route HeaderMatchMethod

An object representing the method and value to match the header value sent with a request. Specify one match method.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Exact" : String,
  "Prefix" : String,
  "Range" : MatchRange,
  "Regex" : String,
  "Suffix" : String
}

YAML

  Exact: String
  Prefix: String
  Range: 
    MatchRange
  Regex: String
  Suffix: String

Properties

Exact

The header value sent by the client must match the specified value exactly.

Required: No

Type: String

Update requires: No interruption

Prefix

The header value sent by the client must begin with the specified characters.

Required: No

Type: String

Update requires: No interruption

Range

The object that specifies the range of numbers that the header value sent by the client must be included in.

Required: No

Type: MatchRange

Update requires: No interruption

Regex

The header value sent by the client must include the specified characters.

Required: No

Type: String

Update requires: No interruption

Suffix

The header value sent by the client must end with the specified characters.

Required: No

Type: String

Update requires: No interruption

AWS::ElasticBeanstalk::ConfigurationTemplate SourceConfiguration

An AWS Elastic Beanstalk configuration template to base a new one on. You can use it to define a AWS::ElasticBeanstalk::ConfigurationTemplate resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "ApplicationName" : String,
  "TemplateName" : String
}

YAML

  ApplicationName: String
  TemplateName: String

Properties

ApplicationName

The name of the application associated with the configuration.

Required: Yes

Type: String

Minimum: 1

Maximum: 100

Update requires: No interruption

TemplateName

The name of the configuration template.

Required: Yes

Type: String

Minimum: 1

Maximum: 100

Update requires: No interruption

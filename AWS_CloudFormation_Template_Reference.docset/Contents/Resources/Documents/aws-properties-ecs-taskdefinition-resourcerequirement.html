AWS::ECS::TaskDefinition ResourceRequirement

The ResourceRequirement property specifies the type and amount of a resource to assign to a container. The only supported resource is a GPU. For more information, see Working with GPUs on Amazon ECS in the Amazon Elastic Container Service Developer Guide

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Type" : String,
  "Value" : String
}

YAML

  Type: String
  Value: String

Properties

Type

The type of resource to assign to a container. The only supported value is GPU.

Required: Yes

Type: String

Allowed Values: GPU

Update requires: Replacement

Value

The number of physical GPUs the Amazon ECS container agent will reserve for the container. The number of GPUs reserved for all containers in a task should not exceed the number of available GPUs on the container instance the task is launched on.

Required: Yes

Type: String

Update requires: Replacement

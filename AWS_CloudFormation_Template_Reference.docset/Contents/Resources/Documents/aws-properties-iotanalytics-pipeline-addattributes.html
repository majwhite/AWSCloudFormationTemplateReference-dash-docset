AWS IoT Analytics Pipeline AddAttributes

The AddAttributes property type specifies additional attributes to be added to a message for an AWS IoT Analytics pipeline.

AddAttributes is a property of the Activity property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Attributes" : JSON object,
  "Name" : String,
  "Next" : String
}

YAML


Attributes: JSON object
Name: String
Next: String

Properties

Attributes

A list of objects that map an existing attribute to a new attribute in a message.

Required: No

Type: JSON object

Update requires: No interruption

Name

The name of the 'addAttributes' activity.

Required: No

Type: String

Update requires: No interruption

Next

The next activity in the pipeline.

Required: No

Type: String

Update requires: No interruption

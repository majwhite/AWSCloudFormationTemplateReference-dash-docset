AWS::EC2::SpotFleet InstanceIpv6Address

Describes an IPv6 address.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Ipv6Address" : String
}

YAML

  Ipv6Address: String

Properties

Ipv6Address

The IPv6 address.

Required: Yes

Type: String

Update requires: Replacement

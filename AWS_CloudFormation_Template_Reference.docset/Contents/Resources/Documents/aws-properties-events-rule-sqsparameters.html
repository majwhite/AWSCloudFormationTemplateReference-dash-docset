Amazon CloudWatch Events Rule SqsParameters

The SqsParameters property type specify the custom parameter to be used when the target is an Amazon SQS FIFO queue.

SqsParameters is a sub-property of the Target property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "MessageGroupId" : String
}

YAML

MessageGroupId: String

Properties

For more information, including constraints, see SqsParameters in the Amazon CloudWatch Events API Reference.

MessageGroupId

The FIFO message group ID to use as the target.

Required: Yes

Type: String

Update requires: No interruption

AWS::GameLift::GameServerGroup Tags

A label that can be assigned to a GameLift resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Tags" : [ Tag, ... ]
}

YAML

  Tags: 
    - Tag

Properties

Tags

A label that can be assigned to a GameLift resource.

Required: No

Type: List of Tag

Update requires: No interruption

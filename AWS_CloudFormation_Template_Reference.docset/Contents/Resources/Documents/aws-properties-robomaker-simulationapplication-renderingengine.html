RoboMaker SimulationApplication RenderingEngine

The RenderingEngine property type specifies the rendering engine for an AWS RoboMaker simulation application.

RenderingEngine is a property of the AWS::RoboMaker::SimulationApplication resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Name" : String,
  "Version" : String
}

YAML

"Name" : String,
Version: String

Properties

Name

The name of the rendering engine.

Required: Yes

Type: String

Update requires: No interruption

Version

The version of the rendering engine.

Required: Yes

Type: String

Update requires: No interruption

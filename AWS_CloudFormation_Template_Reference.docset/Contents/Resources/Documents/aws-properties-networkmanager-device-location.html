AWS::NetworkManager::Device Location

Describes a location.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Address" : String,
  "Latitude" : String,
  "Longitude" : String
}

YAML

  Address: String
  Latitude: String
  Longitude: String

Properties

Address

The physical address.

Required: No

Type: String

Update requires: No interruption

Latitude

The latitude.

Required: No

Type: String

Update requires: No interruption

Longitude

The longitude.

Required: No

Type: String

Update requires: No interruption

AWS::IoTWireless::WirelessDevice OtaaV10x

OTAA device object for create APIs for v1.0.x

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "AppEui" : String,
  "AppKey" : String
}

YAML

  AppEui: String
  AppKey: String

Properties

AppEui

The AppEUI value, with pattern of [a-fA-F0-9]{16}.

Required: Yes

Type: String

Update requires: No interruption

AppKey

The AppKey value. TThe AppKey value is The AppKey is a secret key, which you should handle in a similar way as you would an application password. You can protect the AppKey value by storing it in the AWS Secrets Manager and use the secretsmanager to reference this value.

Required: Yes

Type: String

Update requires: No interruption

AWS::WAFv2::RuleGroup LabelSummary

List of labels used by one or more of the rules of a AWS::WAFv2::RuleGroup. This summary object is used for the following rule group lists:

  • AvailableLabels - Labels that rules add to matching requests. These labels are defined in the RuleLabels for a Rule.

  • ConsumedLabels - Labels that rules match against. These labels are defined in a LabelMatchStatement specification, in the rule statement.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Name" : String
}

YAML

  Name: String

Properties

Name

An individual label specification.

Required: No

Type: String

Update requires: No interruption

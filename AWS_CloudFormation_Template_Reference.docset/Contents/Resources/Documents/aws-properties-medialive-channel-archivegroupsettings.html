AWS::MediaLive::Channel ArchiveGroupSettings

Identifies this output group as an Archive output group, and configures all the parts of the output group except for its outputs. This element belongs to OutputGroupSettings.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Destination" : OutputLocationRef,
  "RolloverInterval" : Integer
}

YAML

  Destination: 
    OutputLocationRef
  RolloverInterval: Integer

Properties

Destination

A directory and base filename where archive files should be written.

Required: No

Type: OutputLocationRef

Update requires: No interruption

RolloverInterval

Number of seconds to write to archive file before closing and starting a new one.

Required: No

Type: Integer

Update requires: No interruption

AWS::NetworkFirewall::RuleGroup Flags

A set of flags, used for the Flags and Masks settings in a AWS::NetworkFirewall::RuleGroup TCPFlagField.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Flags" : [ String, ... ]
}

YAML

  Flags: 
    - String

Properties

Flags

A set of flags, used for the Flags and Masks settings in a AWS::NetworkFirewall::RuleGroup TCPFlagField.

Valid values are the following: FIN | SYN | RST | PSH | ACK | URG | ECE | CWR

For example, [ "FIN", "RST" ].

Required: No

Type: List of String

Update requires: No interruption

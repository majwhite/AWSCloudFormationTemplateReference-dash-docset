AWS::EC2::Instance ElasticInferenceAccelerator

Specifies the Elastic Inference Accelerator for the instance.

ElasticInferenceAccelerator is a property of the AWS::EC2::Instance resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Type" : String
}

YAML

  Type: String

Properties

Type

The type of elastic inference accelerator. The possible values are eia1.small, eia1.medium, and eia1.large.

Required: Yes

Type: String

Update requires: No interruption

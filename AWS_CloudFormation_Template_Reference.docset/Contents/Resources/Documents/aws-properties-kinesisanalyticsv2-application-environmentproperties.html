Amazon Kinesis Data Analytics Application EnvironmentProperties

The EnvironmentProperties property type specifies execution properties for a Java-based Kinesis Data Analytics application.

EnvironmentProperties is a property of the ApplicationConfiguration property.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "PropertyGroups" : [ PropertyGroup, ... ]
}

YAML


PropertyGroups: 
  - PropertyGroup

Properties

PropertyGroups

Describes the execution property groups.

Required: No

Type: List of PropertyGroup

Update requires: No interruption

AWS::MediaLive::Channel OutputDestinationSettings

The OutputDestinationSettings property type specifies Not currently supported by AWS CloudFormation. for an AWS::MediaLive::Channel.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "PasswordParam" : String,
  "StreamName" : String,
  "Url" : String,
  "Username" : String
}

YAML

  PasswordParam: String
  StreamName: String
  Url: String
  Username: String

Properties

PasswordParam

key used to extract the password from EC2 Parameter store

Required: No

Type: String

Update requires: No interruption

StreamName

Stream name for RTMP destinations (URLs of type rtmp://)

Required: No

Type: String

Update requires: No interruption

Url

A URL specifying a destination

Required: No

Type: String

Update requires: No interruption

Username

username for destination

Required: No

Type: String

Update requires: No interruption

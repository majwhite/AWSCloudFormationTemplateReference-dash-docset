AWS::OpsWorks::Instance TimeBasedAutoScaling

Describes an instance's time-based auto scaling configuration.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Friday" : {Key : Value, ...},
  "Monday" : {Key : Value, ...},
  "Saturday" : {Key : Value, ...},
  "Sunday" : {Key : Value, ...},
  "Thursday" : {Key : Value, ...},
  "Tuesday" : {Key : Value, ...},
  "Wednesday" : {Key : Value, ...}
}

YAML

  Friday: 
    Key : Value
  Monday: 
    Key : Value
  Saturday: 
    Key : Value
  Sunday: 
    Key : Value
  Thursday: 
    Key : Value
  Tuesday: 
    Key : Value

AWS::IoTWireless::WirelessGateway LoRaWANGateway

LoRaWAN wireless gateway object.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "GatewayEui" : String,
  "RfRegion" : String
}

YAML

  GatewayEui: String
  RfRegion: String

Properties

GatewayEui

The gateway's EUI value, with the pattern ^(([0-9A-Fa-f]{2}-){7}|([0-9A-Fa-f]{2}:){7}|([0-9A-Fa-f]{2}\s){7}|([0-9A-Fa-f]{2}){7})([0-9A-Fa-f]{2})$

Required: Yes

Type: String

Update requires: No interruption

RfRegion

The frequency band (RFRegion) value. Maximum length is 64.

Required: Yes

Type: String

Update requires: No interruption
